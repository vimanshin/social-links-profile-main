## Мой процесс

### Чем сделано

- Семантическую HTML-разметку
- BEM-методологию
- CSS reset и custom properties
- Подключение шрифтов через @font-face
- Flexbox с центрированием по двум осям
- Mobile-first адаптивность
- Псевдоклассы :hover, :focus-visible
- WCAG-доступность
- Медиа-запросы
